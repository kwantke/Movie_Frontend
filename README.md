# 영화 프로젝트

### 기본 구성
- Reactjs + Typescript 언어로 개발.
- 로그인(회원가입), 영화 메인 및 상세(영상), 검색 기능 구현
- Dokerfile 및 Jenkins Pipeline 구성 

### 영화 로그인 화면
 - 로그인 및 회원 가입 기능 구현
 - JWT Token 기반 방식으로 로그인 구현
 - Redux를 활용하여 로그인(Token) 관리
